# ROFI UTILS

### INSTALL

#### Requirement
```
sudo apt install rofi
```
```
git clone git@github.com:oraimagini/rofi.git
cd rofi
cp -r * ~/.config/rofi/  
```

#### or (testing)

#### Option 1
- Powermenu
```
./scripts/pwm.sh
```

#### Option 2
- Control de volumen
```
./scripts/cvm.sh
```

#### Option 3
- Launcher Scripts
```
./scripts/lsr.sh
```

#### Option 4
- Git Status
```
./scripts/gitstatus.sh
```
