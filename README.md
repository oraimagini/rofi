# ROFI UTILS

### INSTALL

#### Requirement
```
sudo apt install rofi
```
```
git clone git@github.com:oraimagini/rofi.git
cd rofi
```

#### Option 1
- Powermenu
```
cd powermenu
./pwm.sh
```

#### Option 2
- Control de volumen
```
cd volumecontrol
./cvm.sh
```

#### Option 3
- Launcher Scripts
```
cd launcherscripts
./lsr.sh
```

#### Option 4
- Git Status
```
cd gitstatus
./gitstatus.sh
```
